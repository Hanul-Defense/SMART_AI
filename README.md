# 사용법

```
deactivate
rm -rf ~/angle-venv
python3.9 -m venv ~/angle-venv
source ~/angle-venv/bin/activate
```

가상환경 설치 및 최신화

```
pip install --upgrade pip
```

가상환경 내의 pip 최신화

```
pip install opencv-python mediapipe numpy
```

필요 패키지 다운

```
python3 angle_calculator.py
```
